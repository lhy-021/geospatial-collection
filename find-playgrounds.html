Find a Suitable Playground Back to main page

The aim of this task was to find suitable sites in 5 suburbs for building a playground. Criteria factored in were: lot area, zoning, slope, flood, and vegetation. The output was a KML file and it was then consumed in Google Earth to create a 3D visualization of the site. Using the latest GIS technology and the right workflow can largely decrease the time needed for BI analysis and streamline the process, helping people to make informed decisions.
