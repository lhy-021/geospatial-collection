GIS Audit on Land Zoning Back to main page

This is relatively complex analysis work, topology in GIS is always hard. In an ideal world, zoning layer should perfectly align with the cadastral layer, if not, some parcels would be applied with a zoning code or control that they shouldn't be. This can pose a serious problem when amplified. This task was to pinpoint these wrongly zoned properties and create a report of them for further assessment.
CSV reports of current land parcels of different zones were generated using SQL queries. Another set of reports were generated using GIS software. The two were then compared to find disputed areas. A map and a spreadsheet would be output at the end of the process for each zoning as a result of the audit.
The entire workflow was automated.
